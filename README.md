## 현재 구현

- 모바일 우선 앱 쉘 (360px~430px)
- 홈, 검색, 팔기, 마이마린, 더보기 하단 탭
- 더미 매물 30개 (제트스키 5 / 모터보트 8 / 낚시보트 6 / 요트 6 / RIB 5)
- 카테고리/가격/지역/인증/운송/정렬 필터 + 실시간 검색
- 매물 상세: 다중 이미지 갤러리, 진단 리포트, AI시세, 이력, 비용 안내
- sticky CTA: 문의하기 / ⇄ 비교담기 / 방문예약
- **비교함**: 최대 4대 나란히 비교, 유리한 값 자동 강조
- 찜, 비교함, 최근 본 매물, 판매 등록 localStorage 저장
- 판매 등록 7단계 플로우 (Self/Pro, 가이드 사진 슬롯 10개)
