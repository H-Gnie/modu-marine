# 모두의 마린

중고 보트, 제트스키, 요트, 낚시보트 등 수상레저 탈것 거래 앱 MVP입니다.
KB차차차의 정보 구조를 참고하되 브랜드/문구/화면을 복제하지 않고, 수상레저 도메인에 맞게 재구성했습니다.

## 실행

Node나 빌드 도구가 필요 없는 정적 웹앱입니다.

```bash
cd /home/hj/modu-marine
python3 -m http.server 4173
```

브라우저에서 `http://localhost:4173`을 엽니다.

## 현재 구현

- 모바일 우선 앱 쉘
- 홈, 검색, 팔기, 마이마린, 더보기 하단 탭
- 더미 매물 데이터 30개
- 카테고리/가격/지역/인증/운송/정렬 필터
- 매물 상세, 진단 리포트, 문의/방문/운송 CTA mock
- 찜, 비교함, 최근 본 매물, 판매 등록 localStorage 저장
- 셀프 등록/전문가 진단 판매 7단계 mock 플로우
